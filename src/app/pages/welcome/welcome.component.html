ngx-echarts

ngx-echarts is an Angular(ver >= 2.x) directive for ECharts (ver >= 3.x)

GitHub Documents

## Install ```bash # if you use npm npm install echarts -S npm install ngx-echarts -S # or if you use yarn yarn add echarts yarn add ngx-echarts ```
## Import ```typescript import {{'{'}} NgxEchartsModule } from 'ngx-echarts'; @NgModule({{'{'}} imports: [ ..., NgxEchartsModule.forRoot({{'{'}} echarts: () => import('echarts') }) ], }) export class AppModule {{'{'}} } ```
